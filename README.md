# Hangman Game (Python)

A simple command-line Hangman game built with Python.

## Features

* Randomly selects a word from a predefined list.
* Displays hidden letters using underscores.
* Allows the player to guess one letter at a time.
* Reveals all matching letters when a correct guess is made.
* Tracks guessed letters to prevent duplicate guesses.
* Validates user input (only one letter is allowed).
* Gives the player 6 lives.
* Displays win and lose messages.

## Technologies Used

* Python 3
* `random` module

## How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python hangman.py
```
